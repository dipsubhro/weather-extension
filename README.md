# 🌦️ Weather Report Extension

A simple and responsive browser extension that provides real-time weather updates based on your current location or searched city.

## 🚀 Features

- 🌍 Get current weather by geolocation or manual city input
- ⏱️ Real-time temperature, humidity, wind speed, and weather condition
- 📍 Location auto-detect using browser's geolocation API
- 🌗 Responsive and clean UI with light/dark mode support
- ⚡ Fast and lightweight, perfect for quick weather checks

## 🛠️ Built With

- HTML, CSS, JavaScript
- OpenWeatherMap API (or any other API you used)
- [Manifest V3](https://developer.chrome.com/docs/extensions/mv3/) (for Chrome extensions)
